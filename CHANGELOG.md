# Changelog

## [0.1.0] - 2026-05-07

### Features

- IGeometryService and SHiPGeometryService with GeoModel and Geant4 support
- IMagneticFieldService and SHiPMagneticFieldService with mp-units quantities
- Field map visualisation tool (plot_field_map)
- Unit tests for geometry and magnetic field services
